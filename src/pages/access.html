Microsoft Access

O Microsoft Access, também conhecido por MSAccess, é uma base de dados relacional que permite o desenvolvimento rápido de aplicações que envolvem tanto a modelagem e estrutura de dados como também o interface a ser utilizado pelo utilizador. O desenvolvimento da estrutura de dados dá-se de forma muito intuitiva, bastando que o utilizador possua conhecimentos básicos em modelagem de dados e lógica de programação.
