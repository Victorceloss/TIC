Microsoft SharePoint Designer 2007

Microsoft SharePoint Designer 2007, é um editor de HTML e design especializado gratuito na web para criar ou modificar sites do Microsoft SharePoint, fluxos de trabalho e páginas web. É uma parte de produtos da família Microsoft. Antigamente, era uma parte da família Microsoft Office 2007, mas nunca foi incluído em qualquer um dos pacotes do Microsfot Office. SharePoint Designer 2007 é o sucessor do Microsoft FrontPage. Era inicialmente um produto de software comercial. Em 31 de março de 2009, no entanto, SharePoint Designer 2007 foi disponibilizado como um freeware.
